# Rock-Paper-Scissor
https://main--animated-longma-382cfe.netlify.app/
## Description
This is a simple web-based Rock, Paper, Scissors game built using HTML, CSS, and JavaScript. Test your luck and decision-making skills by playing this classic hand game against the computer.

## Features
- Play Rock, Paper, Scissors against the computer.
- Interactive and responsive user interface.
- Keep track of your score.

## Technologies Used
- HTML
- CSS
- JavaScript
